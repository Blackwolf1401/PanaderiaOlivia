# Olivias Panadería & Pastelería — React (Vite)

This project is a direct React port of your static HTML/CSS, preserving the original layout and classes.

## Run locally
```bash
npm install
npm run dev
```
Open the printed local URL.

## Assets
Place your images under `public/img/` keeping the same filenames used in the HTML (e.g. `public/img/kuchen-trocito.png`).

## Notes
- Font Awesome is loaded via the kit in `index.html`.
- Global styles are in `src/style.css` (copied from your original `style.css`).
